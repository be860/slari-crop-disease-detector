# Shared Layer (`lib/shared/`)

This directory contains resources, components, models, and helper classes shared across multiple feature modules.

## Directory Structure
- `widgets/`: Reusable UI components (e.g., custom primary buttons, loading indicators, custom input fields).
- `models/`: Shared data models (e.g., User entity, ScanResult entity).
- `extensions/`: Dart extension methods on framework or standard SDK classes (e.g., BuildContext extensions, String extensions).
