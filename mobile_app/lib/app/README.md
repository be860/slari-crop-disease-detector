# App Layer (`lib/app/`)

This directory contains application-wide configurations, root widget setup, themes, routing, and global constants.

## Directory Structure
- `app.dart`: The root `MaterialApp` configuration using GoRouter and custom Material 3 themes.
- `routes.dart`: Centralized navigation configuration using GoRouter.
- `theme.dart`: App colors, typography, styles, and Material 3 ThemeData definition.
- `constants.dart`: Global constants like spacing, animation durations, and local storage keys.
