# Core Layer (`lib/core/`)

This directory contains framework-independent code, system services, and utilities that form the backbone of the application.

## Directory Structure
- `services/`: Interfaces and implementations for external system services (e.g., local database service, camera service, local AI inference service).
- `helpers/`: Non-visual helper methods and classes.
- `utils/`: Common utilities (e.g., logger, formatting, date-time helper).
- `errors/`: Custom error classes, exceptions, and failure models.
